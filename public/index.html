YouTube Info MCP Server

This is a Model Context Protocol (MCP) server that fetches YouTube video information.

MCP Endpoint

POST /mcp

Available Tool

get-youtube-video-info - Fetches YouTube video metadata and optional subtitles.

Parameters

  • url (string, required) - YouTube video URL
  • includeSubtitles (boolean, optional) - Whether to include subtitles, defaults to false

Usage with Claude Desktop

{
  "mcpServers": {
    "youtube-info": {
      "command": "npx",
      "args": [
        "mcp-remote@next",
        "https://<your-site>.netlify.app/mcp"
      ]
    }
  }
}
